Retro snake é um jogo desenvolvido pelos alunos Ana Beatriz e Artur Rizzi que tem a intenção de retomar uma nostalgia do famoso jogo da cobrinha que tinha no celular da Nokia nos anos 2000. Nós aceitamos o desafio de desenvolver esse jogo porém, com novas funções agora a cobra pode enfrentar um chefão!

Para instalar o nosso jogo siga os seguintes passos:

  • Caso não tenha o VSCode instalado, baixe ele e garanta que o Python 3.10 esteja instalado também;
  • Dentro do VSCode, certifique-se de que você possui a biblioteca Pygame instalada, caso não tenha, digite o seguinte comando no terminal: pip install pygame;
  • Após isso, extraia o arquivo .zip instalado e abra ele pelo VSCode. Clique no arquivo play.py e aperte F5 para rodar o jogo;
  • Caso deseje apenas clonar o repositório, clique aqui
  • Abra o terminal e digite python play.py
